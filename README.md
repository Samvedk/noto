# NOTO 📝  
### by Sam.k  

A premium digital notebook designed to replace physical notebooks.

---

## 🚀 Features

- Minimal Apple-like UI  
- Folder system (Grade → Subject → Notebook → Page)  
- Offline-first (no internet required)  
- Smart delete system (type DELETE to confirm)  
- Touch-first design (no keyboard dependency)  
- Extracurricular tagging system for flexible organization  

---

## 🧠 Product Thinking

NOTO is designed around clarity and intent.

- Instead of creating separate sections for extracurriculars,  
  a lightweight tagging system is used to keep the interface clean  
  while still allowing flexible categorization.

- Instead of instant deletion, NOTO uses intentional friction  
  (type DELETE to confirm) to prevent accidental data loss.

The goal is to reduce clutter while maintaining control.
